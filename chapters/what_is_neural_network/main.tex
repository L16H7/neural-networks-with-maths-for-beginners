\chapter{Neural Network ဆိုတာဘာလဲ?}

Neural network ဆိုတာ သင်္ချာနည်းအရ ပြောမယ်ဆိုရင် data ကိုသင်ယူပြီး အဖြေတွေထုတ်ပေးတဲ့ function တစ်ခုပဲ ဖြစ်ပါတယ်။
function ဆိုတဲ့ အတိုင်း တစ်ခုခုထည့်ပေးလိုက်ရင် တစ်ခုခုပြန်ထုတ်ပေးပါမယ်။ neural network function ထဲကိုထည့်ပေးမယ် data ကို input လို့
ခေါ်ပြီး $x$ နဲ့ရေးလေ့ရှိပါတယ်။
neural network ကို ပြန်ထုတ်ပေးမယ့် အဖြေကိုတော့ $y$ နဲ့ရေးပါတယ်။

$$
f(x) = y
$$
သင်္ချာနည်းအရ neural network ကို ဒီလို အရိုးအရှင်းဆုံး မြင်ကြည့်လို့ရပါတယ်။

အမြင်သာဆုံး ဥပမာပေးရရင် ခွေးပုံနဲ့ကြောင်ပုံနဲ့ ခွဲပြောပေးတဲ့ neural network ပေါ့။ neural network ထဲကို ဝင်မယ့် input $x$ ကို ခွေးတစ်ကောင်ရဲ့ပုံ၊ ဒါမှမဟုတ် ကြောင်တစ်ကောင်ရဲ့ပုံပေါ့။ ပြန်ထုတ်ပေးမယ့် အဖြေ $y$ က "ခွေး" သို့ "ကြောင်" ဆိုတဲ့ အဖြေပေါ့။

$$
\begin{aligned}
f(\text{ခွေးပုံ}) &= \text{ခွေး} \\
f(\text{ကြောင်ပုံ}) &= \text{ကြောင်}
\end{aligned}
$$

အခေါ်အဝေါ်တွေ အရင်မိတ်ဆက်ကြရအောင်။ neural network လို့ နည်းပညာအရခေါ်ပေမယ့် တကယ်လက်တွေ့မှာတော့ model လို့ပဲခေါ်ပါတယ်။
Machine learning model, deep learning model, AI model အစရှိသည်ဖြင့် တစ်မျိုးစီကွဲပြားကြပေမယ့် အားလုံးကို model လို့ခေါ်လဲမှန်ပါတယ်။

AI ကို သင်ပေးတာကို training လုပ်တယ်လို့ခေါ်ပါတယ်။ AI ကို သင်ပေးတဲ့ data ကို train dataset လို့ခေါ်ပြီး AI ဘယ်လောက်တတ်သွားလဲဆိုတာ
ပြန်ပြီး စစ်ဆေးမယ့် data ကို test dataset လို့သတ်မှတ်ကြပါတယ်။
